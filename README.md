## ✉ 커밋 컨벤션
|제목|내용|
|------|---|
|init|작업 세팅 커밋 (패키지 설치 등)|
|feat|기능 추가 및 변경 (화면 영향 o)|
|design|only css 변경|
|fix|기존의 버그 수정|
|refactor|더 좋은 코드 개선 (화면 영향 x)|
|etc|문서 작성 or 그 외 커밋(주석, 개행 등)|
